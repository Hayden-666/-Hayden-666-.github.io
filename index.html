Rules:
the binary puzzle shown below consists a 6 *6 grid, and each cell in the grid can onluy contain a 0 or 1.
You can select different incomplete grid and complete the puzzle according to the following rules:

  • You are not allowed to have more than two adjacent squares with same numbers
  • Each row and column must contain the same number of 0s and 1s. In the grid there must be three 0s and 1s in each row and column.
  • Each row must not be the same as any other row, and each column must not the same as any other column.
